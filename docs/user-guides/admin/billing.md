# Admin: Billing

## Hourly rates

1. Go to **Billing** (`/billing`).
2. View existing rates (workspace, per-user, or per-project).
3. Add a rate:
   - **Project rate** — applies to all billable time on that project (highest priority).
   - **User rate** — applies to that member across projects when no project rate exists.
4. Members can also have a **default hourly rate** on their user profile as a fallback.

Rates can have an **effective from** date for history.

## Billing summary

1. On the billing page, pick a **date range**.
2. Review total hours, billable hours, and total amount (USD label in v1).

Optional filters by project or member match the reporting API.
