# Admin: Dashboard and team live

## Dashboard

1. Go to **Dashboard** (`/dashboard`).
2. Choose a date range preset (e.g. last 7, 30, or 90 days).
3. Review widgets: hours, revenue, utilization, pending approvals, live presence, and more.

### Rearrange widgets

1. Click **Arrange dashboard**.
2. Drag widgets to reorder.
3. **Done** — saves for your session.
4. **Done & save as default** — also sets the layout used when you click **Reset layout**.

Totals use the same billing rules as exports and the billing summary page.

## Team live

1. Go to **Team** (`/team`).
2. See member status: **Active**, **Idle**, **Break**, or **Offline**.
3. Filter by status or search by name/project.
4. Data refreshes every few seconds.

Use this for standups or to confirm someone forgot to stop a timer.

## Team management vs team live

| Page               | Focus                                           |
| ------------------ | ----------------------------------------------- |
| `/team-management` | Roles, invites, hours this week, remove members |
| `/team`            | Live timer/presence status right now            |
